Установить StampScaner на Android

Канал: Internal beta · Платформа: Android 7.0+ (API 24+)
Это ранняя сборка. Распознавание сейчас отвечает демо-результатом, чат с Postman через 1–2 секунды покажет ответ-заглушку. Реальное распознавание включается с подключением backend-Worker — об этом будет отдельное уведомление в приложении. Коллекция, темы, переключение языков, переходы между экранами — работают полноценно.

Шаг 1 · Скачать APK

Текущая сборка — v1.0 (52,5 МБ):

↓ Скачать StampScaner-v1.0.apk
Наведите камеру телефона

Альтернативно: открыть с телефона страницу релиза и нажать на файл StampScaner-v1.0.apk в разделе Assets.

Шаг 2 · Проверить подпись (для параноиков)

Это необязательно, но полезно если APK получили не с GitHub, а через мессенджер: подделать SHA-256 в пути доставки нельзя, и совпадение хэша с публичным — гарантия что файл не подменили.

В терминале Android (через Termux) или на компьютере выполнить:

sha256sum StampScaner-v1.0.apk

Должно вывести:

8ca865a8a5dc916ec1fa3e8b3aabb79c03a1b6389b7c980bb6541dcb69ffbdb5

Хэш каждой новой сборки публикуется в release notes на GitHub и обновляется здесь.

Шаг 3 · Разрешить установку из неизвестных источников

Android по умолчанию блокирует установку APK мимо Google Play. Это нормальная мера безопасности. Для StampScaner её нужно временно ослабить.

  1. Открыть Настройки → Приложения → Специальный доступ (или Дополнительно) → Установка из неизвестных источников.
  2. Найти приложение, через которое скачали APK (Chrome, Telegram, Файлы…) и разрешить ему устанавливать APK.
  3. На Android 13+ путь немного другой: Настройки → Конфиденциальность → Специальный доступ → Установка приложений из внешних источников.

Шаг 4 · Установить APK

Открыть скачанный app-release.apk. Android покажет диалог «Установить это приложение?». Нажать Установить. Спустя 5–15 секунд появится диалог «Готово».

Шаг 5 · Первый запуск

На главном экране найти иконку StampScaner (бургунди марка с буквой S). При первом открытии приложение попросит разрешения:

  • Камера — для распознавания марок. Можно отказать, тогда сканер будет работать только из галереи.
  • Хранилище / Фото — для выбора фото из галереи и сохранения локальной коллекции.

Эти разрешения можно потом изменить в настройках Android. StampScaner никуда фото не отправляет, пока пользователь не нажмёт «Распознать» или «Заявка на сертификат».

Шаг 6 · Что попробовать

  • Главная. Hero-блок, четыре быстрых действия, пустая лента «недавно распознанных».
  • Сканер. Нажать «Сделать снимок» → разрешить камеру → щёлкнуть любую почтовую марку. Через 1–2 секунды появится экран «Результат распознавания» с шкалой уверенности.
  • Коллекция. На экране результата нажать «Добавить в коллекцию». Открыть вкладку «Коллекция» — марка появится с баннером общей оценки. Свайп влево удаляет запись.
  • Postman. Перейти на вкладку Postman, написать любое сообщение про марку. Через 1–2 секунды появится ответ-заглушка (реальный ответ — после подключения Worker).
  • Сертификат. На экране результата нажать «Заказать сертификат», выбрать Pre-cert или Premium, заполнить e-mail, отправить заявку.
  • Профиль. Переключить тему Light/Sepia/Dark и язык RU/EN/AR — интерфейс перерисовывается мгновенно.

Решение частых проблем

«Приложение не установлено»

Чаще всего — на телефоне уже стоит другая версия с тем же package id (например, debug-сборка). Удалить старую через Настройки → Приложения → StampScaner → Удалить и попробовать снова.

«Этот APK не подписан»

На Android 14+ release-APK обязательно должен быть подписан. Текущая сборка подписана debug-ключом — для internal beta этого достаточно, но Google может предупредить. Это нормально, продолжать установку.

Камера не запускается

Открыть Настройки → Приложения → StampScaner → Разрешения → включить Камера. Если разрешение уже включено — перезапустить приложение.

Краш при открытии

Возможные причины: устройство младше Android 7.0 (мы не поддерживаем) или закончилось место на диске. Логи можно собрать командой adb logcat | grep stampscaner и прислать на aslankaa@yandex.ru.

Удаление

Любое из:

  • Долгое нажатие на иконку → Удалить.
  • Настройки → Приложения → StampScaner → Удалить.

Удаление чистит коллекцию из локальной SQLite-базы. Сделать резервную копию пока невозможно — это появится в Спринте 4 вместе с облачной синхронизацией.
